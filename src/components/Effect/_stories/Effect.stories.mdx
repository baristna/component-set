# Effect
#### Effect is a wrapper component that handles css transitions

<br /><br />

## Usage
```jsx
import { Effect } from 'component-set'

<Effect in={state}>...content...</Effect>
```

<br /><br />

## Types

## Events
<Story name="Events">
  <div>
    <EventsExample timeout={3000} />
  </div>

```jsx
<Effect
  in={state}
  onEnter={() => setState('enter')}
  onEntering={() => setState('entering')}
  onEntered={() => setState('entered')}
  onExit={() => setState('exit')}
  onExiting={() => setState('exiting')}
  onExited={() => setState('exited')}
>
  <Box>Fade</Box>
</Effect>
```
