# Flex

##### Quickly manage the layout, alignment, and sizing of grid columns, navigation, components, and more with a full suite of responsive flexDiv utilities.

`Div` component supports all of the <a href='https://styled-system.com/api/#flexbox'>flexbox</a> props

<br /><br />

## Usage

```jsx
import { Div } from 'component-set'

<Div display='flex'>
  /** Content here **/
</Div>
```

<br /><br />

## Direction
default is `row`
<Story name='Direction'>
  <div>
    <Div display='flex'>
      {['row', 'column'].map(property => (
        <Div display='flex' key={uuid()} p={2} flex='0 0 calc(100% / 6)'>
          <Div display='flex' bg={colors['primaryLightest']} height={150} width={'100%'} flexDirection={property}  flexWrap='wrap'>
            {[1,2,3].map(() =>
              <Div display='flex' key={uuid()} m={1} bg={colors['quaternary']} height={30} width={30} flex='0 0 calc((100% / 3) - 16px)' />
            )}
          </Div>
        </Div>
      ))}
    </Div>
  </div>
</Story>

```jsx
<Div display='flex' flexDirection='row'>[Content]</Div>
<Div display='flex' flexDirection='column'>[Content]</Div>
```
