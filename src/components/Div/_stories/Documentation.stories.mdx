# Div

#### The Box component serves as a wrapper component for most of the CSS utility needs.

<br />

<Story name='Overview'>
  <div>
    <Div
      width={0.2}
      padding={2}
      margin={0}
      bg='primary'
      color={'white'}
      boxShadow={'sm'}
    >
      Div
    </Div>
  </div>
</Story>

<br /><br />

## Usage

```jsx
import { Div } from 'component-set';

<Div>
  /** Content here **/
</Div>
```
